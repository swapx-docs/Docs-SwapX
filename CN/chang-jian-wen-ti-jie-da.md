# ❓常见问题解答

## ❓ 常见问题 (FAQ)

本部分提供关于 SwapX 去中心化交易所 (DEX) 常见问题的快速解答。

### 🚀 核心产品与功能

| 问题                     | 回答                                                                                        |
| ---------------------- | ----------------------------------------------------------------------------------------- |
| **什么是 SwapX？**         | SwapX 是一个基于 **XONE 公链**构建的原生**去中心化交易所 (DEX)**。它基于 Uniswap V2/V3 协议，支持代币兑换、流动性提供和法币交易功能。   |
| **SwapX 属于哪个区块链生态系统？** | SwapX 是部署在 **XONE Chain** 上的 DEX，与 XOC 代币、USDH 稳定币和 BVI 行为激励机制等生态模块高度集成。                  |
| **SwapX 支持哪些资产？**      | SwapX 当前支持 XOC, WXOC, USDH, PANDE, PHX, T1, PANDA, PNUTX, TEST, NUTX, PEPE 等。未来将接入更多生态代币。 |
| **如何在 SwapX 中兑换代币？**   | 访问 swapx.exchange，点击\*\*“兑换 (Exchange)”\*\*，选择代币对并输入数量，授权后确认交易即可。                         |
| **如何调整 SwapX 的滑点设置？**  | 点击兑换界面的**设置按钮**，自定义滑点容忍度（如 0.1%、0.5%、1%）以控制交易偏差。                                          |
| **SwapX 支持多链吗？**       | SwapX 当前支持 **XONE 主网**，**ETH、BSC 和 Tron** 的支持将很快开放。                                       |

### 🛡️ 去中心化与安全性

| 问题                    | 回答                                                                                                  |
| --------------------- | --------------------------------------------------------------------------------------------------- |
| **SwapX 是一个去中心化平台吗？** | 是的，SwapX 是**完全去中心化**的。所有交易都通过链上智能合约执行，无需信任第三方。用户始终掌控自己的私钥和资产。                                       |
| **SwapX 会托管资产吗？**     | **不会**。SwapX 不托管资产，您始终通过个人钱包与智能合约进行交互。资产安全由您的私钥控制。                                                  |
| **SwapX 会收集用户隐私信息吗？** | **不会**。SwapX 不存储用户身份信息或钱包私钥，仅保留必要的链上数据交互信息用于服务优化。                                                   |
| **SwapX 需要实名注册吗？**    | **不需要**。SwapX 不需要账户注册或实名注册，使用钱包即可进行交易。                                                              |
| **SwapX 支持哪些钱包？**     | SwapX 支持 **MetaMask、TokenUp、Trust Wallet、OKX Wallet、Coinbase Wallet、Bitget、TokenPocket、Phantom** 等。 |
| **如何将钱包连接到 SwapX？**   | 访问 SwapX 官网，点击右上角的\*\*“连接钱包 (Connect Wallet)”\*\* → 选择支持的钱包 → 完成链上授权。                               |

### 💰 流动性提供与费用

#### 费用与奖励

| 问题                       | 回答                                                              |
| ------------------------ | --------------------------------------------------------------- |
| **SwapX 会收取多少交易手续费？**    | SwapX 的默认手续费为 **0.3%**，全部用于奖励流动性提供者。未来可能会引入 **0.05% 的协议费用**。    |
| **向 SwapX 添加流动性会得到什么？**  | 您将获得 **LP Token**，并按比例分享交易手续费。每笔交易您可获得 **0.3% 的佣金分成**，按流动性比例结算。 |
| **SwapX 会推出交易挖矿或激励计划吗？** | **是的**。SwapX 将定期推出交易竞赛、LP 奖励活动、持有补贴等运营计划。                       |

#### 流动性池管理

| 问题                        | 回答                                                                                                       |
| ------------------------- | -------------------------------------------------------------------------------------------------------- |
| **什么是 SwapX 的流动性池？**      | 流动性池是用户提供代币对组成的**智能合约资金池**，用于实现自动做市和代币兑换。                                                                |
| **如何向 SwapX 添加流动性？**      | 前往\*\*“资金池 (Fund Pool)”\*\*页面，选择代币对，输入数量，授权后确认添加即可获得 **LP Token**。                                       |
| **什么是 SwapX 的 LP Token？** | 它是您提供流动性的**凭证**，可用于赎回佣金分红或参与激励活动。                                                                        |
| **SwapX 流动性可以随时移除吗？**     | **可以**。用户可以随时移除流动性。进入\*\*“资金池 (Fund Pool)”\*\* → **“我的持仓 (My Holdings)”**，点击\*\*“移除流动性”\*\*，设置比例并确认钱包操作。 |
| **如何移除 SwapX 的流动性？**      | 前往\*\*“我的持仓”**页面，选择池子，点击**“移除 (Remove)”\*\*，设置比例并确认。                                                     |
| **SwapX 中的无常损失是什么？**      | 当您提供的代币价格大幅波动时，与直接持有代币相比，您可能会损失部分收益。建议选择低波动性的稳定币组合。                                                      |
