# 💰如何获取并创建钱包

## 入门指南：钱包设置

连接钱包是使用 SwapX 的基础步骤，为您提供了对数字资产的去中心化控制权。

### 🔑 如何获取和创建钱包

#### 📱 移动钱包 vs. 💻 桌面钱包：如何选择？

选择移动或桌面钱包取决于您的安全性和访问优先级。请参考以下对比，选择最适合您的钱包类型：

| 特性                    | 移动设备钱包 | 桌面钱包 |
| --------------------- | ------ | ---- |
| **随时随地使用**            | ✅      | ➖    |
| **易于使用**              | ✅      | ➖    |
| **安全性** (与其他应用隔离)     | ➖      | ✅    |
| **可访问性** (快速访问)       | ➖      | ✅    |
| **防损坏/丢失/盗窃** (密钥已备份) | ➖      | ✅    |
| **防断电/断连**            | ✅      | ➖    |

#### 🛠️ SwapX 支持的钱包

SwapX 兼容广泛且安全的流行钱包，确保为所有用户提供灵活性：

* 🦊 **MetaMask**
* ⚪ **TokenUp**
* 🛡️ **Trust Wallet**
* ⚫ **OKX Wallet**
* 🟦 **Coinbase Wallet**
* 👻 **phantom**
* 🟡 **Bitget Wallet**

### ⚠️ 关键安全与设置注意事项

创建和管理钱包时，必须严格遵守安全协议以保护您的资金。这些规则是保护您资产的基石：

| 操作 | 指南                                 |
| -- | ---------------------------------- |
| ✅  | **请务必仅从官方来源下载并安装最新版本。**            |
| ✅  | **请仔细遵循设置指南。**                     |
| ✅  | **请安全备份您的恢复短语（助记词）。**              |
| ❌  | **在任何情况下，绝不与任何人分享您的恢复短语。**         |
| ❌  | **绝不要在您的钱包应用之外的任何网站或应用中输入您的恢复短语。** |

_💡 提示：如果您没有可靠的下载来源，您可以直接通过 SwapX 界面获取经过验证的链接。_

#### 🚀 如何通过 SwapX 获取钱包

如果您想快速开始，SwapX 在连接界面提供了直达支持钱包的链接：

1. 访问 SwapX，点击右上角的 **"Connect Wallet"（连接钱包）** 按钮。
2. 点击 **"Get a Wallet"（获取钱包）** 按钮。
3. 点击您想要获取的钱包旁边的 **"Get Wallet"（获取钱包）** 按钮。
4. 选择您想要使用的钱包类型（移动应用或浏览器扩展程序）并进行安装。

_如果您已经安装了钱包，请前往连接指南了解如何使用。_
